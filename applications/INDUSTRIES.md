        │
        ▼

### 1. MATERIALS SCIENCE
**Status: Active — Fixed point specified**

**Profit point:** BCS theory explained Pb superconductivity at 7.2 K. Question declared closed ~1970s.

**Layers frozen:**
- FCC phase assumption
- Ambient pressure requirement  
- Processing contamination tolerance
- Debye temperature inherited from bulk

**Fixed point:** Simple cubic Pb, a = 3.18 Å, Tc = 295 K

**Why it matters:** Room-temperature superconductivity would eliminate transmission losses in the entire electrical grid. Every wire, every motor, every generator. The energy implications alone are civilization-scale.

### 2. FINANCE / CREDIT
**Status: Active — Implementation in development**

**Profit point:** FICO adopted by lenders and regulators, 1989. System optimized for compatibility, not accuracy.

**Layers frozen:**
- Bureau monopoly on input data
- Static monthly snapshot
- W-2 employment bias
- Algorithm opacity
- Gig economy invisibility

**Fixed point:** Real-time, multi-source, worker-owned financial identity score

**Why it matters:** 59 million gig workers. 45 million credit-invisible Americans. A financial system that cannot see the people in it is not a financial system. It is a gatekeeping mechanism.

**Verification:** See `applications/finance/`

### 3. COMMUNICATIONS
**Status: Active — Theoretical framework developed**

**Profit point:** TCP/IP and electromagnetic propagation accepted as fundamental constraints. Speed of light treated as absolute limit for practical purposes.

**Layers frozen:**
- Signal propagation assumption
- Sequential send-receive model
- Classical channel dependency
- Infrastructure compatibility requirement

**Fixed point:** Recursive synchronization via shared seed — coordination without transmission

**Why it matters:** Every millisecond of latency has economic cost. Financial markets. Remote surgery. Autonomous vehicles. Space exploration. The assumption that communication requires propagation has never been fully tested against alternatives.

**Verification:** See `applications/communications/`
